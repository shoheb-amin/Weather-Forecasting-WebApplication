# Weather Forecasting Web Application

A responsive weather forecasting web application that provides real-time weather updates using external weather APIs. Users can search for any city to view current weather conditions including temperature, humidity, wind speed, and weather status.

## Features

- Search weather by city name
- Real-time weather updates
- Displays temperature, humidity, and wind speed
- Dynamic weather icons
- Responsive user interface
- API integration for live weather data
- Clean and user-friendly design
